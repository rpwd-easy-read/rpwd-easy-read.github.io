# RPwD Act 2016 — Easy Read Guide (web app)

All 102 sections of India's Rights of Persons with Disabilities Act
2016 in Easy Read: plain language, one idea at a time, an illustration
for every concept, with the official Section numbers kept so a reader
can cite the law in a complaint.

Created and maintained by **Deepa Palaniappan**, accessibility
professional (CPACC). Copyright and licence terms: see
[LICENSE.md](LICENSE.md). The Easy Read adaptation, illustrations
(AI-generated by the author), code, and build pipeline are her original
work, begun April 2026.

## What it does

- 102 sections across 17 colour-coded chapters, each with what the
  section says, when to use it, and an illustration
- Search across every section
- Read-aloud on every section (browser text to speech, no server)
- How to complain, who can help, and your rights pages
- Progressive Web App: installs to the home screen, works fully
  offline after first load, no login, no tracking, no backend

## Accessibility

WCAG 2.1 AA floor with Easy Read and COGA practice throughout: semantic
HTML, alt text, visible focus (GOV.UK-style yellow), large tap targets,
Big text toggle, screen reader announcements, system fonts, rem units,
no animation.

## Running it

Static files only. Open `index.html` over any web server (service
workers need http, not file://):

```
python -m http.server 8080
```
